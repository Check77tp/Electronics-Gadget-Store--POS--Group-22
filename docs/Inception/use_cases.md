# High-Level Use Cases

## 01: Process Sale
**Primary Actor:** Cashier  
**Description:** The cashier scans products, calculates total amount, processes payment, and prints receipt.

## 02: Manage Inventory
**Primary Actor:** Manager  
**Description:** The manager adds, updates, or removes products from the system.

## 03: Generate Sales Report
**Primary Actor:** Manager  
**Description:** The manager views sales summaries and performance reports.
